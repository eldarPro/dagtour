# Настройка Supabase

## 1. Создание таблиц

Откройте Supabase Dashboard → SQL Editor и выполните скрипт:

```
supabase/schema.sql
```

## 2. Заполнение начальными данными

В SQL Editor выполните:

```
supabase/seed.sql
```

## 3. Настройка аутентификации

В Supabase Dashboard → Authentication → Providers:
- Включите Email провайдер
- При необходимости настройте другие провайдеры (Google, Telegram и т.д.)

## 4. Структура таблиц

### Публичные справочники:
- `houses` — дома для аренды
- `cars` — автомобили для аренды
- `tours` — туры

### Пользовательские данные:
- `user_cars` — автомобили пользователей
- `user_houses` — дома пользователей
- `profiles` — профили пользователей

## 5. Безопасность (RLS)

Все таблицы защищены Row Level Security:
- Публичные таблицы (houses, cars, tours) — чтение для всех
- Пользовательские таблицы — доступ только к своим данным
